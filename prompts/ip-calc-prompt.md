# Промт для расчёта IP-адресации

Скопируй и замени значения из своего задания.

```text
Ты сетевой инженер. Рассчитай IPv4-адресацию для демонстрационного экзамена по 09.02.06.

Домен: au-team.irpo
Базовый приватный диапазон: 192.168.0.0/16

Требования:
1. VLAN HQ-SRV: <VLAN>, нужно <N> usable-адресов или формулировка из задания.
2. VLAN HQ-CLI: <VLAN>, нужно <N> usable-адресов или формулировка из задания.
3. VLAN MGMT: <VLAN>, нужно <N> usable-адресов или формулировка из задания.
4. BR-SRV: нужно <N> usable-адресов.
5. WAN ISP-HQ: 172.16.1.0/28.
6. WAN ISP-BR: 172.16.2.0/28.
7. GRE/IPIP tunnel: подобрать /30.

Нужно выдать:
1. Таблицу VLAN → subnet → mask → gateway → DHCP range.
2. Таблицу устройств: ISP, HQ-RTR, BR-RTR, HQ-SRV, HQ-CLI, BR-SRV.
3. Таблицу DNS A/PTR записей.
4. Объяснение, почему выбрана каждая маска.
5. Проверку, что адреса не пересекаются.
6. Если VLAN больше 255, объяснить, как корректно отразить его в 192.168.* сети.

Не выдумывай. Все расчёты покажи явно.
```
