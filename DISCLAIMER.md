# Disclaimer

Этот репозиторий предназначен только для учебной подготовки к демонстрационному экзамену по 09.02.06.

Он не является официальным экзаменационным материалом, эталонным решением или гарантированным ответом на реальный вариант ДЭ.

## Перед использованием

Проверь:

- свой вариант задания;
- реальные VLAN;
- реальные IP-сети;
- имена сетевых интерфейсов;
- дистрибутив ОС;
- доступные пакеты;
- пути к конфигурационным файлам;
- наличие Additional.iso;
- наличие дисков перед RAID.

## Пароли

В публичном репозитории используется плейсхолдер:

```text
CHANGE_ME_FROM_TASK
```

Заменяй его на пароль из своего экзаменационного задания или лабораторного стенда.

## Опасные команды

Особенно внимательно проверяй команды:

```bash
wipefs
mdadm --create
mkfs.ext4
iptables -F
iptables -P
```

Они могут удалить данные, пересобрать RAID, сломать сеть или отрезать удалённый доступ.
